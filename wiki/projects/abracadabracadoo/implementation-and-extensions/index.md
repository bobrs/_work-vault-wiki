## Working Read

The development directive frames the protocol as a clean C++ reference library with local storage, test coverage, and a mock server for loop verification.

The addenda extend the protocol into witnessable loops, loop-local timebases, and conditional deniability. Those three changes define the main extension boundary of the family.
