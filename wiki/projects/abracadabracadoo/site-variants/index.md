## Working Read

These bundles show the public site becoming a static Astro surface with a docs gateway, app surface, and public assets bridge. The `v7` and `v8` variants are near duplicates in structure, but the README wording and gateway emphasis show a real iteration path rather than noise. The earlier `v1` through `v3` bundles show the same public surface before the later docs/footer polish.

This page stays separate from the protocol pages so the public delivery surface can evolve without being confused for the formal message spec or implementation directive.
