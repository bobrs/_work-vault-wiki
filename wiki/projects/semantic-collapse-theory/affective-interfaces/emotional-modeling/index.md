## Working Read

This is the affective substrate: emotion modeled as loop topology and then translated into interface layers.

The page is doing more than naming feelings. It is preserving the idea that emotion can be represented as a structured state transition without losing the fact that it is lived, directional, and embodied.

That makes this cluster a bridge between raw affect and the more interpretive SCT pages. It belongs next to the therapeutic and temporal branches because it helps explain how a state is stabilized before anyone tries to reason about it.

If the branch grows, the split should follow a difference between internal emotional structure and interface representation.
