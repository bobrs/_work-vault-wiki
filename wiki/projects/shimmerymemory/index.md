# Shimmery Memory

This branch collects the ShimmeryMemory site snapshots and the essay corpus that feeds them.

It reads as a public memory site with a glyph field, a publishing contract, and a large essay rail. The site snapshots show the surface evolving; the essays are the durable content substrate; the seed manifest package keeps the curated origin set visible.

## Current Shape

- 2 archived site snapshots.
- 56 published essays in the latest snapshot.
- 1 seed document-set manifest package with 5 listed items.
- 2 nested support pages organize the set.

## Working Read

This is a public essay site, not a loose pile of notes.

The site snapshots hold the public surface and the glyph experience. The essay corpus is the actual content line. The publishing contract makes the output format explicit so the project can keep absorbing new essays without blurring source and presentation.
