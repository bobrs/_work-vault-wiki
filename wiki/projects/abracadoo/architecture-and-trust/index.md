## Working Read

The roadmap says Abracadoo should start as a HumanKey-first app: one-way Acquaintances first, local secret storage behind a vault boundary, and relationship state derived from completed loops rather than asserted by a flag.

The manifesto translates that into public language. Trust should have a path, recognition can be one-way, and machine participation still has to sit inside accountable loops.
