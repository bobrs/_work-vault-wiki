How it works

1. Create

Capture your style, boundaries, and values in a small context file (JSON or text).

2. Host

Keep it on GitHub, Google Drive, or your own site. Context² just links to it.

3. Connect

Use extensions or integrations so apps fetch your latest context automatically.

4. Control

Share, expire, or revoke access at any time. Consent is built in.
