# Symbola licensing/source note

Requested asset: Symbola font by George Douros / Unicode Fonts for Ancient Scripts.

The exact font binary could not be fetched automatically from this environment, so this package includes a source note and license note rather than the font file itself.

Known source references used for this note:
- Font Library: https://fontlibrary.org/en/font/symbola
- GitHub mirror: https://github.com/aur-archive/ttf-symbola
- Backup repo: https://github.com/ChiefMikeK/ttf-symbola

Fonts in this site are offered free for any use; they may be opened,

fonts designed by George Douros"
