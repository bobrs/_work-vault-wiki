# Abracadoo ephemeral contacts patch

This patch updates the Abracadoo front page to introduce the no-usernames / ephemeral-contact realization without losing the warm hometown-trust register.

## Included

- `abracadoo/index.html` and `abracadoo/styles.css`: updated homepage files.
- `ephemeraltrust/index.html`: a one-page seed for EphemeralTrust.com.

## Main changes

- Reframes contacts as local relationship labels, not usernames.
- Replaces username-ish preview text (`Your HumanKey: jon`, `Connections`) with local key / local labels.
- Adds a dedicated “No usernames” section.
- Adds EphemeralTrust as the plain-language field guide and links it from the builder/footer areas.
- Keeps Abracadoo warm, neighborly, and protocol-serious rather than turning the homepage into a philosophy wall.
